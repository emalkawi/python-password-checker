# python-password-checker
# Password Strength Checker + Hash Generator

This program checks password strength and JWT secret security using Python.

## Features
- Validates password strength
- Checks JWT Secret strength
- Generates SHA-256 hash for strong passwords

## What is Hashing?
Hashing is a process that converts data into a fixed string value. 
It is used to store passwords securely instead of plain text. 
Even if someone accesses the database, they cannot read the original password.
